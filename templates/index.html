Welcome to the Student Exam Performance Predictor

Predicting math scores using machine learning techniques

Project Overview

This project predicts a student's math score based on various input parameters such as gender, ethnicity, parental education level, lunch type, test preparation course, and other factors. The model is trained using machine learning techniques to ensure accurate predictions.

Exploratory Data Analysis (EDA)

Before building the prediction model, extensive Exploratory Data Analysis (EDA) was performed. Various charts and visualizations were created to understand the data better:

  • Bar charts
  • Violin plots
  • Scatter plots
  • Correlation heatmaps

Machine Learning Workflow

The machine learning pipeline was structured into different stages:

  • Data Ingestion: Loading and preprocessing raw data
  • Data Transformation: Applying scaling, encoding, and imputation
  • Model Training: Training multiple regression models and selecting the best

Prediction Pipeline

The trained models and preprocessing techniques were saved as preprocessor.pkl and model.pkl. These are used in the prediction pipeline located in the pipeline folder.

Models Used

Several regression models were tested, and the best-performing model was selected based on evaluation metrics:

  • Linear Regression
  • Decision Trees
  • AdaBoost Regressor
  • XGBoost Regressor
  • CatBoost Regressor
  • Gradient Boosting Regressor
  • K-Neighbors Regressor
  • Random Forest Regressor

Feature Engineering & Optimization

To improve model performance, various preprocessing techniques were applied:

  • Standard Scaling
  • Data Imputation
  • Column Transformation
  • One-Hot Encoding
  • Normalization
  • Hyperparameter Tuning

Deployment with Flask

The project is deployed using Flask, providing a user-friendly interface for predictions. Users can input their details, and the model will return a predicted math score.
